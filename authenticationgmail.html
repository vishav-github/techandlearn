Authentication from Gmail Server to Checker System

Follow these steps to authenticate your Gmail server with the Checker System.

Step 1: Enable 2-Step Verification

Make sure you have enabled 2-Step Verification on your Gmail account. Read more here.

Step 2: Generate App Password

Go to the App Passwords page and create a password for Checker System.

Generate App Password

Step 3: Save the Generated Password

After generating the password, save it for future use.

Step 4: Use the Password in Checker System

Use this password when configuring email settings in Checker System.
