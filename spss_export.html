SPSS Export

When exporting to SPSS, ensure that only the codes of answers and free text are included in the custom export.

Steps to Export

  • Export the required data using the SPSS export option.
  • Locate the exported files and follow the steps shown below.

Instructions

  1. Download the exported files.
  2. Copy and paste the exported folder into your working directory.
  3. Run the SPSS software to generate the .SAV file.

After completing these steps, SPSS will generate a .SAV file with all the label values from the questionnaire.
