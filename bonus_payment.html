Automated Bonus / Penalty

  • If the report is submitted within the "Time till to give default bonus on review," the system will automatically update the shopper's payment.
  • If the editor approves the report and it meets the criteria, the system will apply the bonus.
  • For automatic bonuses, the system checks the report's completion time and applies the bonus accordingly.

Bonus Calculation

  • Fixed Bonus: A fixed amount is granted if submitted on time.
  • Percentage Bonus: A percentage of the survey payment is given if submitted on time.

Example: If the survey payment is 350 and the percentage bonus is 15%, the automatic bonus should be 52.50.
