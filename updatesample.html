Update Sample Column Titles when Using Auto Import

Steps for changing column titles of a sample survey and updating Auto Import.

Step 1: Edit Sample Titles

Go to Management > Sample, find the sample that needs updates, and click "Edit Sample Titles" to change the column titles as required.

Step 2: Update Auto Import Settings

Once column titles are changed, the system clears all selected columns for auto import. Re-select them by going to:

Management > Sample > Sample Auto Import Profile

Click the profile's time to edit.

At General > Import Columns, select the columns you will use for import.

Step 3: Run Auto Import

The system will automatically import files from the FTP folder. If you want to run the import immediately after updating column titles, click "Run Now" at:

Management > Sample > Sample Auto Import Profile
